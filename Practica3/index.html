Arduino Electrónica Attiny

- Resumen -

Este curso consta de 13 ejercicios para arduino-attiny aprendiendo a programarlos de forma adecuada, los archivos necesarios se encuentran en sus respectivas secciones.

- Introducción -

Para la realización de estos ejercicios es necesario tener conocimientos previos en programación arduino, utilizando attiny85 es posible crear circuitos funcionales de menor tamaño para realizar funciones específicas e implementarlas.

- Materiales -

Arduino UNO, software Arduino, Attiny85, archivos descargables de cada práctica, cables macho-macho, leds rojos, resistencias de 100 ohm, botones.

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar que utilizaremos arduino como programador para el atinny. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, como los delay, los cuales nos permitiran separar las respuestas en milisegundos.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el "tools" y después de "upload", con upload hacemos que el código lo ejecute attiny.
