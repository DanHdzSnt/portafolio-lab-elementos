Arduino Electrónica Attiny Sensores

- Resumen -

Este curso consta ejercicios para utilizar sensores digitales, analógicos e inteligentes

- Introducción -

Para la realización de estos ejercicios es necesario tener conocimientos previos en programación arduino y descargar las librerias correspondientes para cada sensor.

- Materiales -

Arduino UNO, software Arduino, Attiny85, archivos descargables de cada práctica, cables macho-macho, sensor de presión barométrica, sensor de flama, sensor oxígeno, sensor ultrasónico.

Construcción

Para la construcción de cada ejercicio se deberá seguir el diagrama de outputs de cada sensor y conectarlos a arduino con base al código.

- Desarrollo -

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar que utilizaremos arduino. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, como los delay, los cuales nos permitiran separar las respuestas en milisegundos. Algunos códigos requieren librerias que no posee arduino por defecto, por lo que se deberán descargar.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el "tools" y después de "upload", con upload hacemos que el código lo ejecute arduino.

- Conclusiones -

Los sensores permiten construir dispositivos electrónicos avanzados que pueden facilitar la vida cotidiana de los humanos y ayudarlos en diversas labores. Es indudable su influencia, con esta práctica podemos darnos una idea los dispositivos que contienen al menos un sensor y como puede funcionar.
