Arduino Electrónica Attiny

- Resumen -

Este curso consta de 17 ejercicios para arduino-attiny aprendiendo a programarlos de forma adecuada y controlar distintos componentes electrónicos para realizar acciones determinadas, los archivos necesarios se encuentran en sus respectivas secciones.

- Introducción -

Para la realización de estos ejercicios es necesario tener conocimientos previos en programación arduino y completar los cursos anteriores, utilizando attiny85 es posible crear circuitos funcionales de menor tamaño para realizar funciones específicas e implementarlas.

- Materiales -

Arduino UNO, software Arduino, Attiny85, archivos descargables de cada práctica, cables macho-macho, leds rojos, resistencias de 100 ohm, botones, led RGB, motor CC, servomotores, fuente de voltaje, caimanes, neopixel, tira con 5 leds y buzzer.

Construcción

Para la construcción de cada ejercicio se cuenta con un diagrama el cuál se deberá conectar de la manera indicada. Para los ejercios que requieren el uso de una fuente de voltaje, configurar el amperaje en 5 Ameperios y el Voltaje en 5V, asegurarse de conectar correctamente VCC y GND del protoboard a la fuente.

- Desarrollo -

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar que utilizaremos arduino como programador para el atinny. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, como los delay, los cuales nos permitiran separar las respuestas en milisegundos. Algunos códigos requieren librerias que no posee arduino por defecto, por lo que se deberán descargar, tal es e caso de la libreria neopixel, al usar esta libreria se debera configurar el reloj interno en 8 MHz para poder usarlo.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el "tools" y después de "upload", con upload hacemos que el código lo ejecute attiny.
