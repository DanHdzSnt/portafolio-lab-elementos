Arduino Electrónica Servomotores

- Resumen -

Este curso consta de 8 ejercicios para arduino en el que se aprenderá a usar los servomotores y que estos trabajen de manera conjunta o independiente, los archivos necesarios se encuentran en sus respectivas secciones.

- Introducción -

Para la realización de estos ejercicios es necesario contar con conocimientos básicos de arduino o bien, completar el curso anterior para comprender mejor el funcionamiento de los revomotores.

- Materiales -

Arduino UNO, software Arduino, archivos descargables de cada práctica, cables macho-macho, leds rojos, resistencias de 100 ohm, servo motores, potenciómetros y una fuente de energía.

- Desarrollo -

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar el tipo de arduino que vamos a utilizar, en nuestro caso Arduino UNO. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, como los delay, los cuales nos permitiran separar las respuestas en milisegundos y las funciones para controlar los servos y sus parámetros.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el botón "upload" ubicado junto al botón que usamos para compilar el código, con upload hacemos que el código lo ejecute arduino.
