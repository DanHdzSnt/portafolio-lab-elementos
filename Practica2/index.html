Arduino Electrónica

- Resumen -

Este curso consta de 18 ejercicios para tener un mejor entendimiento sobre arduino a nivel básico, en la sección de descargas podrás encontrar el código arduino de cada práctica así como su diagrama para recrearlo.

- Introducción -

Para estos ejercicios no es necesario tener conocimientos previos, de momento solo se requerirá seguir los pasos indicados.

- Materiales -

Arduino UNO, software Arduino, archivos descargables de cada práctica, cables macho-macho, leds rojos, resistencias de 100 ohm, display de 7 segmentos y botones.

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar el tipo de arduino que vamos a utilizar, en nuestro caso Arduino UNO. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, como los delay, los cuales nos permitiran separar las respuestas en milisegundos.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el botón "upload" ubicado junto al botón que usamos para compilar el código, con upload hacemos que el código lo ejecute arduino.
